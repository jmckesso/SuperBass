# SuperBass.java

Yup


### Installing

Clone repo

```
$ git clone https://github.com/jmckesso/SuperBass.git
```

## Running

Change into cloned directory

```
$ cd SuperBass
```

Compile the program

```
$ make build
```

Run SuperBass and be amazed

```
$ make run
```
